Satellite Image Change Detector
Agent: When user draws a bounding box on the map and selects a date range, the agent fetches satellite imagery for that area at multiple time points, aligns images, runs a pixel-level difference analysis, classifies the type of change detected (deforestation, construction, flooding), and alerts if the change exceeds a set threshold.
Frontend: Map-based Bounding Box Selector — draw a region and set a date range; "Detect Changes" triggers the agent | Image Timeline Viewer — agent-fetched images displayed as a scrubable slider | Change Heatmap Overlay — agent-computed difference map showing where change occurred | Change Classification Table — agent-categorised events with date, type, and affected area in sq km | Alert Subscription Modal — user sets a change-size threshold; agent sends an email when exceeded
