Personalised Podcast Recommender
Agent: When user rates five seed podcasts, the agent fetches their metadata, extracts topics from episode descriptions, builds a preference model, queries podcast databases for unseen episodes matching that model, ranks them by relevance, and creates a scheduled listening plan.
Frontend: Seed Rating Page — five podcast cards shown; user swipes to rate, triggering the agent | Recommended Episodes Feed — agent-ranked episodes with artwork, topic tags, and match percentage | Listening Queue — reorderable list built by the agent, showing total estimated listening time | Preference Map — interactive topic bubble chart generated by the agent showing what the user likes
