OSINT Domain Intelligence Tool
Agent: When user inputs a domain or organisation name, the agent enumerates public GitHub repositories, fetches WHOIS registration records, searches for exposed services using public internet scanning data, aggregates all findings, and compiles a structured intelligence report highlighting potential security exposures.
Frontend: Target Input Form — domain or org name and scope checkboxes; "Investigate" triggers the agent | Discovery Graph — agent-built entity map showing org, repositories, employees, and exposed services | Findings Table — agent-categorised results by source with severity tags | Exposure Risk Score Card — agent-computed overall risk with a breakdown by finding type | Report Export — agent-compiled PDF/JSON report; user clicks "Download Report"
