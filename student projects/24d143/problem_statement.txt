Smart Energy Usage Optimiser
Agent: When user imports smart meter readings, the agent fetches the dynamic electricity tariff schedule, factors in weather-driven heating and cooling demand, identifies peak consumption hours, suggests moving discretionary appliances to low-tariff time slots, and projects the monthly savings.
Frontend: Meter Data Import Page — CSV upload or API key entry; upload triggers the agent | Usage Heatmap Calendar — agent-processed hour-by-day grid showing consumption intensity | Tariff Overlay Chart — agent-combined usage bars and tariff rate line | Appliance Scheduler — drag appliances to low-tariff slots; each move triggers the agent to recalculate projected savings | Savings Projection Dashboard — agent-estimated monthly saving displayed as a running total
