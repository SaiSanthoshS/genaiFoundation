Smart Public Transit Journey Planner
Agent: When user inputs an origin and destination, the agent fetches available route options from transit data, predicts delay probability for each route using historical patterns, calculates the fare, checks live vehicle occupancy where available, and re-routes automatically if a disruption is detected mid-journey.
Frontend: Journey Input Form — origin, destination, and departure time; "Plan Journey" triggers the agent | Route Options List — agent-ranked options showing duration, fare, delay probability, and CO₂ estimate | Live Journey Map — agent-tracked vehicle position on the selected route | Delay Alert Panel — agent-detected disruption with a re-route suggestion generated in real time | Departure Alert Scheduler — user sets a reminder; agent calculates and sends notification at the right time
