Smart Library Catalogue & Reader
Agent: When user searches for a book, the agent queries an open library database, scores readability level, checks whether a free digital edition exists, retrieves the full text if available, and when the user finishes, recommends the next read based on themes in books they have completed.
Frontend: Book Search Page — title, author, or genre search; submitting triggers the agent | Book Detail Page — agent-fetched cover, description, and readability level with free/paid indicator | Online Reader — chapter navigation and highlight tool; user highlights trigger the agent to save bookmarks | Progress Dashboard — agent-tracked pages read, reading speed estimate, and streak | Recommendations Shelf — agent-selected next books based on completed reading history
