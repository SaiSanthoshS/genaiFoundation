Biodiversity & Wildlife Tracker
Agent: When user selects a location and date range, the agent queries public wildlife observation databases, retrieves species information and habitat data, checks conservation status for every species found, maps sighting density, and sends an alert when an endangered species is recorded in the user's area.
Frontend: Location & Date Filter Page — map click and date range; "Find Wildlife" triggers the agent | Sightings Map — agent-placed clustered markers colour-coded by species class | Species Detail Page — agent-fetched photo, habitat description, and IUCN conservation status badge | Biodiversity Dashboard — agent-calculated species count chart by class (mammals, birds, reptiles, etc.) | Alert Setup Page — user selects species of interest; agent sends a notification when a new sighting is logged nearby
