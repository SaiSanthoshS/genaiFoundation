Code Snippet Search Engine
Agent: When user types a natural-language programming query, the agent searches public code repositories and developer Q&A forums, extracts relevant code blocks, detects the programming language of each, ranks results by community approval and recency, and annotates each snippet with a plain-English explanation.
Frontend: Natural Language Search Bar — query submission triggers the agent | Snippet Results List — agent-ranked cards with syntax-highlighted code, language badge, and source attribution | Language Filter Sidebar — user narrows results; each filter re-triggers the agent | Snippet Detail Page — full agent-annotated code with a copy button and a link to the original thread | Saved Snippets Collection — bookmarked results; agent flags if a snippet's source thread is updated
