Epidemic Spread Simulator
Agent: When user selects a disease and region, the agent loads real case data, calibrates the transmission model to match actual historical spread, runs multiple intervention scenarios (no action, masks, vaccination), and produces animated projections showing how each scenario plays out over 180 days.
Frontend: Disease & Region Selector — dropdown and map click; "Run Simulation" triggers the agent | Parameter Input Panel — user adjusts vaccine rate and intervention date; each change triggers the agent to re-run | Animated Spread Map — agent-generated day-by-day choropleth the user can scrub through | SIR Curve Chart — Susceptible / Infected / Recovered lines produced by the agent | Scenario Comparison Table — agent-computed peak cases and total affected per intervention option
