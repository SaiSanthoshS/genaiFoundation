## Data Flow

1. **User uploads resume** вҶ’ PDF/DOCX text extraction вҶ’ Groq LLM parsing вҶ’ Structured JSON
2. **ATS analysis** вҶ’ Resume JSON вҶ’ Groq LLM scoring вҶ’ Score + feedback
3. **Job search** вҶ’ Parallel API calls to 3+ boards вҶ’ Dedup вҶ’ Store in DB
4. **AI matching** вҶ’ Candidate vector вҶ’ ChromaDB similarity вҶ’ Groq LLM scoring вҶ’ Ranked results
5. **Cover letter** вҶ’ Resume + Job data вҶ’ Groq LLM вҶ’ Personalized letter
6. **Interview prep** вҶ’ Resume + Job вҶ’ Groq LLM вҶ’ Questions + STAR answers
7. **Tracker** вҶ’ User-managed Kanban вҶ’ SQLite persistence
